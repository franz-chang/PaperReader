# MiniOneRec: An Open-Source Framework for Scaling Generative Recommendation

## 一句话总结
这篇论文把生成式推荐做成了完整的开源训练框架，并系统验证了这条路线确实存在随模型规模增长而变好的趋势。

## 论文做了什么
作者关注的是一个现实问题：生成式推荐近几年很热，但工业实现大多不开源，外界很难验证它到底能不能像大语言模型一样吃到规模化红利。这篇论文因此提出了 MiniOneRec，一个从语义 ID 构建、监督微调到强化学习后训练都打通的开源框架，用来复现和研究生成式推荐。

## 核心方法
它先用 Residual Quantized VAE 给物品构造 Semantic ID，再用 Qwen 系列自回归模型去预测下一个物品的 ID 序列。作者还设计了一套很轻量的后训练流程，重点包括全流程的 SID 对齐，以及带约束解码和混合奖励的推荐强化学习，以同时兼顾排序效果和推荐多样性。

## 结果与意义
实验表明，模型从 0.5B 扩到 7B 时，训练和测试损失都持续下降，说明生成式推荐确实有明显的参数扩展收益。后训练之后，排序准确率和候选多样性也继续提升。可以把这篇论文看成生成式推荐能不能做大、怎么做大的一份开源答案。
