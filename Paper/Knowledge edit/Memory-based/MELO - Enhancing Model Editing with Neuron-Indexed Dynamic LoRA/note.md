# MELO: Enhancing Model Editing with Neuron-Indexed Dynamic LoRA

## 一句话总结
这篇论文把知识编辑做成了一个动态激活的 LoRA 插件系统，用很小的额外参数实现持续编辑。

## 论文做了什么
作者认为，现有编辑器往往难以同时兼顾成功率、局部性、持续编辑能力和计算开销，而且对不同主干模型的适配也不够灵活。MELO 的目标是提供一种足够轻、又能覆盖多种编辑场景的插件式方案。

## 核心方法
MELO 用 neuron-indexed dynamic LoRA 来存放编辑，每次查询时根据内部向量数据库检索到相关编辑索引，再动态激活对应的 LoRA block。这样模型不需要每次都改主权重，而是像调用局部适配器一样加载编辑。

## 结果与意义
实验显示，MELO 在顺序编辑任务上取得了很强表现，同时训练参数量和计算成本都更低。它的意义在于把知识编辑和参数高效微调结合起来，给出了一条工程上很有吸引力的折中路线。
