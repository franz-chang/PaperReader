# In-Context Editing: Learning Knowledge from Self-Induced Distributions

## 一句话总结
这篇论文试图让编辑后的模型既学到新知识，又不因为过拟合而把语言质量和泛化能力一起拖垮。

## 论文做了什么
作者认为，很多基于微调的知识编辑方法之所以副作用大，是因为它们把目标答案当成 one-hot 监督信号，优化过猛，容易过拟合。ICE 的目标是把编辑过程变得更像上下文学习，让模型以内化新知识的方式完成更新。

## 核心方法
ICE 的做法是对齐模型在有上下文和无上下文条件下的输出分布，不再只盯着单个目标 token。这样模型学习到的是一种上下文分布上的一致性，而不是被强行拉到某个尖锐答案点上。

## 结果与意义
论文从准确率、局部性、泛化性和语言质量四个维度评估，显示 ICE 比常规梯度式编辑更稳。它的意义在于把 in-context learning 的思想真正引入了知识编辑优化目标。
