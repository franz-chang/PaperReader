# WISE: Rethinking the Knowledge Memory for Lifelong Model Editing of Large Language Models

## 一句话总结
这篇论文的核心观点是：持续知识编辑里，可靠性、泛化性和局部性很难同时满足，除非重新设计知识该存在哪。

## 论文做了什么
作者先分析了两种常见路线：直接改参数的长期记忆，以及基于检索激活的工作记忆。结论是，这两条路线各有硬伤，单独使用时都会落入一个不可能三角：可靠性、泛化性和局部性无法兼得。

## 核心方法
WISE 因此提出双参数记忆结构，把预训练知识放在主记忆里，把编辑知识写到侧记忆里，再训练一个 router 来判断当前查询该走哪条路径。为了解决持续编辑冲突，它还设计了知识分片和合并机制，让不同编辑尽量分布在不同子空间。

## 结果与意义
实验表明，WISE 在问答、幻觉纠正和分布外设定下都优于此前方法，并在持续编辑场景里更好地平衡了成功率与副作用。它的关键价值在于把问题从怎么改参数，提升到编辑知识该放在哪种记忆结构里。
