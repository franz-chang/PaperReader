# Can We Edit Factual Knowledge by In-Context Learning?

## 一句话总结
这篇论文想回答一个很直接的问题：如果不改参数，只靠上下文示例，能不能把模型的事实性回答改掉。

## 论文做了什么
作者注意到，传统知识编辑往往依赖梯度更新，这在超大模型或黑盒 API 场景下成本很高甚至根本做不到。于是他们系统研究了另一条路线：只靠 in-context learning，能否实现有意义的事实编辑。

## 核心方法
IKE 的方法本身并不复杂，重点在于构造合适的演示上下文，让模型在当前会话中倾向于采用新事实。论文对多种 ICL 编辑策略进行了系统比较，并把它们与参数更新型方法做对照。

## 结果与意义
结果显示，IKE 在 GPT-J 上已经能获得接近梯度编辑方法的成功率，同时副作用更小，对无关相似事实和已存知识的干扰也更少。它的重要启发是：编辑并不一定都要写进参数，提示上下文本身也可以是编辑通道。
