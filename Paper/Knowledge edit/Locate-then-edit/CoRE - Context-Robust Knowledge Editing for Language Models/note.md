# Context-Robust Knowledge Editing for Language Models

## 一句话总结
这篇论文指出，很多知识编辑方法只会在裸问题上成功，一旦前面加上上下文就失效；CoRE 的目标就是让编辑结果在真实对话上下文里也稳定成立。

## 论文做了什么
作者认为现有知识编辑评测过于理想化，通常只测编辑后的单个问句，却忽略了真实使用中常见的前置上下文。于是他们先构建了 CHED 这个 benchmark，用来专门测试上下文会不会把模型重新拉回原知识。在这个更贴近实战的设定下，很多已有方法都出现明显退化。

## 核心方法
CoRE 的核心思路是降低编辑知识在不同上下文下的隐藏状态波动，让模型不那么依赖上下文线索来决定是否启用新知识。换句话说，它不是只让目标答案变对，而是让编辑后的表示在不同上下文里更稳。

## 结果与意义
实验表明，CoRE 能显著提升带上下文情况下的编辑成功率，同时不会明显伤害模型整体能力。这篇论文的价值在于把知识编辑从单条问答修补推进到了真实交互环境下是否稳健这个更严肃的问题。
