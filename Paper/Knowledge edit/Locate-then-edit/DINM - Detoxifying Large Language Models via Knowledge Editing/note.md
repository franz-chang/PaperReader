# Detoxifying Large Language Models via Knowledge Editing

## 一句话总结
这篇论文把知识编辑用于模型安全，提出用极少量更新快速压低大模型的有毒输出。

## 论文做了什么
作者关注的是模型去毒问题：与其对整个模型重新做安全训练，不如像改知识一样，直接把容易触发不安全输出的内部参数局部改掉。为此，他们先构建了 SafeEdit 基准，覆盖多类不安全场景，然后系统比较了多种知识编辑方法在去毒任务上的表现。

## 核心方法
论文提出 DINM，用很少的调参步数、甚至只依赖单个样本，就能对模型进行局部去毒。它的目标不是简单压制模型输出，而是更直接地调整与毒性相关的参数，使得模型内部对这些不安全反应模式的表达被真正削弱。

## 结果与意义
实验显示，知识编辑确实可以高效地实现去毒，而且对模型通用能力的副作用相对有限。作者进一步分析发现，传统 SFT 和 DPO 往往更像把毒性压住，而 DINM 更接近把毒性源头改掉。
