# A Unified Framework for Model Editing

## 一句话总结
这篇论文说明 ROME 和 MEMIT 本质上是在优化同一个目标，并在这个统一视角下提出了可做大批量编辑的 EMMET。

## 论文做了什么
过去很多人把 ROME 和 MEMIT 看成两条不同路线：前者擅长单点编辑，后者擅长批量编辑。作者认为这种理解过于表面，于是从优化目标出发重新分析这两种方法，试图找出它们更深层的一致性。

## 核心方法
论文提出 preservation-memorization 统一目标，指出 ROME 相当于在等式约束下做单点编辑，而 MEMIT 则是在更宽松的最小二乘约束下做批量编辑。基于这个统一框架，作者进一步提出 EMMET，用等式约束也能实现大批量编辑。

## 结果与意义
EMMET 在多项指标上与 MEMIT 接近，并能把批量编辑扩展到很大规模。更重要的是，这篇论文的贡献不只是一个新方法，而是把一批经典编辑算法放进同一个理论框架里，帮助大家更清楚地理解它们到底差在哪、其实又有多像。
