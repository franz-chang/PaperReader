# EAMET: Robust Massive Model Editing via Embedding Alignment Optimization

## 一句话总结
这篇论文解决的是大规模批量编辑为什么容易崩的问题，给出的答案是：不同知识项的表示空间没有对齐，所以要先做嵌入对齐再编辑。

## 论文做了什么
很多编辑方法在单次或小批量编辑时表现不错，但一旦扩展到海量知识更新、带复杂上下文、或同一主体对应多条事实时，效果就明显下降。作者把这个现象归因于知识项之间的 embedding 失配，认为这是大规模编辑不稳的根源。

## 核心方法
EAMET 的核心是对 key embedding 和 residual embedding 的空间做显式对齐，让模型在写入大量新知识时，目标表示之间不会彼此冲突。它不是单纯改优化器，而是直接针对知识表示不一致这个结构性问题动手。

## 结果与意义
在六个 LLM 和三个数据集上，EAMET 在大规模编辑场景中持续优于已有方法，编辑一万条事实时仍能保持很高的成功率。这篇论文的重要点是把大规模编辑的瓶颈从算法技巧转向了表示空间质量。
