# Mass-Editing Memory in a Transformer

## 一句话总结
这篇论文把知识编辑从改一条事实推进到同时改成千上万条事实，是大规模参数编辑路线的代表作。

## 论文做了什么
作者认为已有方法虽然能编辑单个事实，但离现实中的知识更新还很远，因为真实系统往往需要一次改很多条知识。MEMIT 的目标就是让模型能在一次参数更新中写入大量新记忆，而不只是局部打一两个补丁。

## 核心方法
它沿着 ROME 的思路，继续把 Transformer 中间层看作可写入的知识记忆位置，然后设计更适合批量写入的更新方式，把多条事实同时注入模型的若干层参数中。核心不是简单累加单条编辑，而是用更系统的方式做批量写入。

## 结果与意义
实验表明，MEMIT 能在 GPT-J 和 GPT-NeoX 上同时编辑数千条事实，规模远超此前方法。这篇论文基本确立了大规模参数内写入这条路线，也是后来很多批量编辑工作比较和改进的基线。
