# PMET: Precise Model Editing in a Transformer

## 一句话总结
这篇论文想把编辑做得更准，核心观点是不能把整个 Transformer 隐状态一锅端地当成 FFN 的记忆值来处理。

## 论文做了什么
很多编辑方法默认认为 Transformer 层隐藏状态就是 FFN 记忆的值，因此直接围绕这个整体表示来优化和更新参数。作者指出这种做法混淆了 MHSA、FFN 和残差连接的作用分工，会导致更新不够精准。

## 核心方法
PMET 先分析 MHSA 和 FFN 在知识提取中的不同角色，认为 MHSA 更像负责一般性的知识抽取模式，而 FFN 更直接承载具体事实。基于这个判断，它会同时优化组件级隐藏状态，但最终只用与 FFN 相关的那部分状态去更新 FFN 权重。

## 结果与意义
在 CounterFact 和 zsRE 上，PMET 达到很强的编辑表现。它的价值在于提醒大家：知识编辑不仅是改哪一层，更是该改表示里的哪一部分，这让编辑从粗粒度走向了更细粒度的结构理解。
