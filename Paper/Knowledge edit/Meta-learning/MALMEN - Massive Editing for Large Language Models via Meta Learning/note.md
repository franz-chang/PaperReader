# Massive Editing for Large Language Models via Meta Learning

## 一句话总结
这篇论文把超网络编辑路线扩展到了大规模批量编辑，重点解决的是很多编辑同时发生时更新如何稳定合成。

## 论文做了什么
作者认为，已有 hyper-network 编辑器虽然灵活，但在同步编辑大量事实时扩展性很差，简单把多个更新累加起来还会互相抵消。MALMEN 的目标就是把 meta-learning 路线也推到大规模编辑场景。

## 核心方法
论文把多条参数位移的聚合写成最小二乘问题，并用 normal equation 来求整体更新。同时，它把超网络和语言模型本体的计算拆开，让两部分都能在有限显存下处理更大的 batch。

## 结果与意义
实验表明，MALMEN 能编辑的事实数量比 MEND 大很多，并且在多种模型和任务上都很强。它的重要价值在于说明：超网络编辑不只是单点修补工具，也可以通过更好的聚合机制走向大规模化。
